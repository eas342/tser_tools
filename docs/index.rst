Welcome to tshirt's documentation!
==================================

.. image:: images/t_shirt_logo.jpg
   :width: 200px
   :alt: T shirt logo
   :align: center

The Time Series Helper & Integration Reduction Tool (tshirt) is a general-purpose tool for time series science. It can:

- Reduce raw data
- Extract Photometry
- Extract Spectroscopy
- Fit models to data
